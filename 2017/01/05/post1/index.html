使用Hexo+GitHub搭建博客

发表于 2017-01-05 | 分类于 Hexo

安装Hexo前提

您必须检查电脑中是否已安装下列应用程序：

Git
Node.js

Windows用户可从以上两个链接下载安装Git和Node.js

使用nvm安装Node.js(若已下载安装可直接跳过该步骤)

安装好Git后，新建一个文件夹作为根目录（文件名不要使用中文），在根目录下右键选择Git Bash，进入命令行终端，然后输入以下命令安装nvm
$ curl https://raw.github.com/creationix/nvm/master/install.sh | sh
又或者
$ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
安装完成后，关掉命令行终端并用Git Bash重新打开，执行下列命令安装Node.js
$ nvm install stable

安装Hexo

安装好Git和Node.js后，新建文件夹（文件名不要使用中文），在文件夹内右键点选Git Bash(相关命令使用linux shell执行，以$开头的代码块都是在命令行终端执行)，进入命令行终端，执行以下命令安装Hexo
$ npm install -g hexo-cli

创建github仓库

新建一个名为的仓库，比如说，如果你的github用户名是test，那么你就新建的仓库（必须是你的用户名，其它名称无效），将来你的网站访问地址就是 http://test.github.io

$ ssh -T git@github.com

输入yes，成功后会看到
Hi xxx! You've successfully authenticated, but GitHub does not provide shell access.
然后进行配置
$ git config --global user.name "username" //github用户名
$ git config --global user.email "xxx@xxx.com" //github注册邮箱
